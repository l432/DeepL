1. Вибрати (за необхідності) папки, де розміщена програма SCAPS ("SCAPS Folder") та  будуть зберігатися результати ("Results Folder") 
2. Встановити режим освітлення (радіо-кнопка)

Для кожної товщини бази та концентрації бору в базі:

3.Встановлюємо значення товщини бази та концентрації бору в базі

4. Кнопка ".scaps file create" - створюємо settings-файли 
  "FeD*.scaps" для розрахунку ВАХ при наявності лише міжвузольного заліза
  "D1Т*.scaps" для розрохунку розподілу дефектів по товщині
  "FeBD*.scaps" для розрахунку ВАХ при пар залізо-бор та міжвузольного заліза

Для кожної температури

5. В SCAPS завантажуємо файл "FeD*.scaps",
    натискаємо "Clear all simulation",
    запускаємо "Calculate: batch",
    зберігаємо *.iv файл.

6."Витягуємо" ВАХ та інформацію  (кнопки "Process .iv file")

7.В SCAPS завантажуємо файл "D1Т*.scaps",
    натискаємо "Clear all simulation",
    запускаємо "Calculate: single shot",
    зберігаємо *.eb файл.

8. Створюємо файли розподілу дефектів по товщині - кнопка "Fe(x) and FeB(x) create".

9. В SCAPS завантажуємо файл "FeBD*.scaps",
    натискаємо "Clear all simulation",
    запускаємо "Calculate: batch",
    зберігаємо *.iv файл.

10."Витягуємо" ВАХ та інформацію  (кнопки "Process .iv file")


11. Зводимо результати: кнопка "all dates.dat convert".
